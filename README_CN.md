# 收支记账APP - 完整实现

这是一个使用ArkTS语言开发的功能完整的收支记账应用，满足所有实验要求。

## 🎯 项目概述

本项目实现了一个专业的个人财务管理应用，支持：
- 📝 灵活的收支记录
- 📊 多维度的数据统计
- 🔍 强大的查询功能
- 🎨 现代化的用户界面

## ✨ 核心功能

### 1️⃣ 记账功能
- 支持收入和支出的分类记录
- 13种预设分类（8种支出 + 5种收入）
- 记录金额、用途、日期、分类等详细信息
- 支持添加、删除、侧滑删除等操作

### 2️⃣ 统计功能
- 今日收支统计
- 本月收支统计
- 按周、月、年进行汇总
- 分类分布统计和百分比计算

### 3️⃣ 查询功能
- 按交易类型筛选（全部/支出/收入）
- 按分类筛选
- 按金额范围筛选
- 按日期范围筛选
- 支持多条件组合查询

### 4️⃣ 图表功能
- 分类分布进度条图表
- 百分比展示
- 详细的统计数据表格
- 月份切换功能

## 📚 使用说明

### 添加记录
1. 点击首页右上角的"+"按钮
2. 选择交易类型（支出/收入）
3. 选择分类
4. 输入金额
5. 输入用途描述
6. 确认日期
7. 点击"保存"

### 删除记录
在流水页面向左滑动记录，点击"删除"按钮

### 查看统计
1. 切换到"统计"标签页
2. 使用左右箭头切换月份
3. 查看各分类的分布情况

### 查询记录
1. 切换到"查询"标签页
2. 设置查询条件
3. 点击"查询"按钮
4. 查看符合条件的记录

## 💾 数据管理

### 存储方式
- 使用HarmonyOS Preferences API
- 本地JSON格式存储
- 自动保存功能

### 数据结构
```typescript
interface AccountRecord {
  id: string;                    // 唯一标识
  type: 'income' | 'expense';   // 收入或支出
  category: string;              // 分类名称
  amount: number;                // 金额
  description: string;           // 用途描述
  date: number;                  // 时间戳
  dateStr: string;               // 日期字符串
}
```
