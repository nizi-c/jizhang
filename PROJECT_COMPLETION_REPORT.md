## 🎯 项目目标

实现一个功能完整的收支记账应用，满足以下要求：

1. ✅ 分类记录日常收入和支出数据
2. ✅ 显示月、日收支流水记录
3. ✅ 按周、月、年进行汇总
4. ✅ 具备友好的用户界面
5. ✅ 支持多条件查询
6. ✅ 界面美观漂亮
7. ✅ 支持统计图表
