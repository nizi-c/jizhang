# 收支记账APP - ArkTS实现

这是一个功能完整的收支记账应用，使用ArkTS语言开发，支持HarmonyOS平台。

## 项目功能

### 1. 首页（Home Tab）
- **今日收支统计**：显示今日收入、支出和结余
- **本月收支统计**：显示本月累计收入、支出和结余
- **最近记录**：显示最近5条交易记录
- **快速添加**：右上角浮动按钮快速添加新记录

### 2. 流水记录（List Tab）
- **完整记录列表**：显示所有交易记录
- **侧滑删除**：向左滑动可删除记录
- **记录详情**：显示分类、金额、用途、日期等信息
- **实时更新**：删除后自动更新统计数据

### 3. 统计图表（Statistics Tab）
- **月份切换**：可选择查看不同月份的数据
- **支出分类分布**：显示支出各分类的占比
- **收入分类分布**：显示收入各分类的占比
- **进度条图表**：直观展示各分类的百分比
- **详细统计**：列出各分类的具体金额

### 4. 查询功能（Query Tab）
- **交易类型筛选**：支持全部、支出、收入三种类型
- **分类筛选**：按选定的分类进行筛选
- **金额范围**：支持设置最小和最大金额范围
- **日期范围**：支持按开始日期和结束日期筛选
- **组合查询**：支持多条件组合查询
- **查询结果**：显示符合条件的所有记录

### 5. 添加记录对话框
- **交易类型选择**：支出或收入
- **分类选择**：可视化分类选择器，包含emoji图标
- **金额输入**：支持小数点输入
- **用途描述**：记录交易的具体用途
- **日期选择**：默认为当前日期，可修改

## 核心数据模型

### AccountRecord（账户记录）
```typescript
interface AccountRecord {
  id: string;                    // 唯一标识
  type: 'income' | 'expense';   // 收入或支出
  category: string;              // 分类名称
  amount: number;                // 金额
  description: string;           // 用途描述
  date: number;                  // 时间戳
  dateStr: string;               // 日期字符串 YYYY-MM-DD
}
```

## 主要功能实现

### 1. 数据持久化
使用HarmonyOS的Preferences API实现本地数据存储，所有记录自动保存到本地。

### 2. 日期处理
提供完整的日期工具函数，支持：
- 日期格式化
- 月份、周、年的提取
- 日期比较和判断

### 3. 数据统计
支持多维度的数据统计：
- 按日期分组
- 按月份分组
- 按分类汇总
- 百分比计算

### 4. 灵活查询
支持多条件组合查询：
- 交易类型
- 分类
- 金额范围
- 日期范围

## 使用说明

### 添加记录
1. 点击首页右上角的"+"按钮
2. 选择交易类型（支出/收入）
3. 选择分类
4. 输入金额
5. 输入用途描述
6. 确认日期（默认为今天）
7. 点击"保存"

### 查看流水
1. 切换到"流水"标签页
2. 查看所有交易记录
3. 向左滑动可删除记录

### 查看统计
1. 切换到"统计"标签页
2. 使用左右箭头切换月份
3. 查看各分类的分布情况
4. 查看详细的金额统计

### 查询记录
1. 切换到"查询"标签页
2. 设置查询条件（类型、分类、金额、日期）
3. 点击"查询"按钮
4. 查看符合条件的记录
